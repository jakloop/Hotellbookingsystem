# Overview:
This an assignment for subject 2 of the year one course Online.
The assignment is to create a hotel booking system.
The console application is written in C#.
The system simulates a hotel booking system with
- different room types (single, double, suite)
- guests (regular, VIP-guests)
- booking system with check-in and check-out
- payment methods implemented with interface
- Elementary tests for core functionality
- A UML diagram of the systems classes and relationships
